# Material-Barriers-to-the-Diffusion-of-the-Magnetic-Field-in-MHD
Datasets and codes for the paper "Material Barriers to the Diffusion of the Magnetic Field".

Datasets are available under the following link: https://polybox.ethz.ch/index.php/s/aT0I5XxCXDgzbIV.
To successfully run the code, please download the folder "data" and put it in the same directory as the jupyter notebooks.
